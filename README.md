# ⏱️ StopClock App - Prodigy Infotech Task

This project is part of the **Prodigy Infotech Internship Program**. It showcases a simple yet functional **StopClock Web App** built using **Streamlit**, designed to help users track elapsed time with start, stop, and reset capabilities.

## 🚀 Deployment

Access the live app here:  
🔗 [StopClock Streamlit App](https://stopclock-i5yobeouercmdku7pk2iwe.streamlit.app/)

## 📌 Features

- Start, Stop, and Reset buttons for time tracking
- Real-time display of elapsed time
- Intuitive and minimalistic UI
- Built with Python and Streamlit

## 🛠️ Technologies Used

- **Python**
- **Streamlit**
- **Datetime module**

```

## 🧠 How It Works

- The app uses Python's `datetime` module to calculate elapsed time.
- Streamlit's session state is used to manage the timer across interactions.
- The UI updates dynamically based on user input.
